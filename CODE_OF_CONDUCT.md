# Code of Conduct

## Our standards
- Be respectful and constructive.
- Assume good intent.
- Focus on technical discussion and outcomes.
- Avoid harassment, insults, and discriminatory language.

## Scope
This applies to project spaces including issues, pull requests, and discussions.

## Enforcement
Project maintainers may remove, edit, or reject comments, commits, code,
issues, and other contributions that violate this Code of Conduct.
